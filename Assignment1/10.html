This is a sample article to demonstrate the use of semantic HTML elements.

Special characters can be included using HTML entities. Here are some examples:

  • & -This is the ampersand symbol and is used to start an HTML entity.
  • < -This is the less than symbol and is used to start an HTML tag.
  • > -This is the greater than symbol and is used to end an HTML tag.
  • © -This is the copyright symbol, represented as © in HTML.
  • ® -This is the registered trademark symbol, represented as ® in HTML.
  • ™ -This is the trademark symbol, represented as ™ in HTML.
